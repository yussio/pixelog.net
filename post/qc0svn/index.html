ユーザーCSSで特定のページを擬似的にアクセス禁止にする

ユーザーCSSで特定のページへアクセスできないようにして、見たくないページを見ない方法。

ページの表示

CSSを当てたページにアクセスしようとすると、下のようなメッセージが表示されます。

Chromeのエラーメッセージの文言をデベロッパーツールで弄ってスクショして作りました。私的な範囲で楽しむ分には大丈夫かと思います。

ユーザーCSSに適用させたい画像のために、わざわざ画像をサーバーにアップロードするのもアレなので、今回はBase64エンコードしてみました。TinyPNGも繰り返し掛けたので圧縮されています。

CSS

アクセスしたくないドメインに下のCSSを適用させればOKです。body要素を丸っとdisplay: noneして、html要素に先ほどの画像をbackgroundに貼り付けます。

body {
    display: none!important;
}

もし、上で指定したドメイン上で例外を設定したい場合は、下のような別のスタイルを作成して上書きしてやればいいと思います。

body {
    display: block!important;
}

html {
    background: none;
}

結局は

この方法はすぐに解除できてしまうので、小細工の域は脱せていません。「見たくないけどどうしても見てしまう」といった類のサイトで禁止にしたい場合は注意が必要です。スマホではそもそもユーザーCSSの当てられるブラウザの数が少ないです。

アクセス禁止にする方法は他にも、hostsファイルでドメインの名前解決を阻害したり、ルーターでIPをフィルタに掛けたりなど、いろいろ考えられますが、他の端末で見るとか、モバイルデータ通信に切り替えるだとか、回避する方法はいくらでもあるので、結局は個人の堅い意志が必要です。

表示をブロックするのはあくまで気休め程度に考えておくのがいいと思います。
